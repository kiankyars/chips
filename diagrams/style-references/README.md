# Image style references

These images define the hand-drawn visual bar for image-model course assets:
confident black ink, tactile white paper, restrained color, strong spatial
explanation, and enough object detail to feel authored rather than templated.

- `euv-light-path.png` — style and explanatory sequencing reference.
- `litho-chain.png` — style and simplified relationship reference.
- `industry-map.png` — **style reference only**. Its connector semantics are not
  course-safe, so it must not be used as the value-chain diagram. The deck uses
  the deterministic `diagrams/rendered/map-*.svg` family instead.

Image-model prompts may use these files for visual language only. Course claims,
labels, numbers, and relationships remain in deterministic Slidev/SVG content.
